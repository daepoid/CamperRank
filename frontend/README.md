## Installation

```bash
$ yarn install
```

## Running the app

```bash
yarn dev
```

VITE_CLIENT_URL="http://[hosting_url]:[client_port]/"

VITE_SOCKET_URL="ws://[hosting_url]:[socket_port]/"
